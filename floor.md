# 会場係

1. タイムキーパー
   - 14分発表、4分質疑としてますが、実質15分発表でよいです。
       - 14分で1回、15分で2回鳴らしてください。
   - この[リンク](https://presentimer.github.io/?limit=15&bell1=14&bell2=15&bell3=0
   )にアクセスして、ベルを鳴らすよう、スマホの音量を上げてください。
   - 発表が始まったら開始、時間内に終わったら停止してください。
     - 発表中はベルが演者に聞こえる範囲にいる方がよかろうと思います。
   - 質疑応答の間にリセットしてください。
   - 事前に練習して音の大きさとか確認してもらえると助かります。
   - スタートし損ねたら、大まかでよいので14分で1回、15分で2回ベルを鳴らしてください。

2. 発表補助
   - 発表者がPCを持ち込むようにしています。接続で手こずっていたらヘルプお願いします。

3. 質疑応答補助
   - A会場は質問者用にマイクが1本あります。質問が手を上げたらマイクを持って行くか、マイクのところに質問者を誘導してください。
   - B, C会場は質問者用マイクがありません。座長のマイクを基本質問者に持っていくようにしてください。状況を見て演者のマイクを持って行ったり、その場でマイクなしで発言してもらってもよいと思います。

4. お菓子、飲み物の補充
   - 2Fにお菓子と1.5, 2Lのペットボトル(緑茶、紅茶、オレンジ、アクエリアス)を置いています。
   - お菓子は紙皿に置いて適宜補充してください。
   - 飲み物もなくなったら補充するようにお願いします
   - ゴミ袋の具合なども注意してください
   - 何か気が付いたことがあれば連絡ください
